PWA Icon Requirements
====================

This directory should contain PWA icons for the Reward Loyalty application.

Required files:
- pwa-192.png (192x192 pixels)
- pwa-512.png (512x512 pixels)

Icon Guidelines:
- PNG format
- Square (1:1 aspect ratio)
- Include ~10% padding for maskable icon support
- Solid background color (not transparent)
- Use app logo/branding

Generation Options:
1. Use realfavicongenerator.net
2. Use maskable.app for maskable icons
3. Export from design tool (Figma, Sketch, etc.)

Current Status:
- Placeholder icons created
- Replace with actual branded icons before production deployment
